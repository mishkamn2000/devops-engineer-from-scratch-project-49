A collection of five math games for brain training.

## Requirements

- Python 3.10 or higher
- uv package manager

## Installation

```bash
git clone https://github.com/mishkamn2000/devops-engineer-from-scratch-project-49.git
cd devops-engineer-from-scratch-project-49
make install
make build
make package-install
